# client

`client` is a web application written in TypeScript.

## Constitution
  - Framework
    - React
  - Language
    - TypeScript
  - Test Tools
    - Enzyme
    - Jest
    
## Prerequisites

- [npm](https://www.npmjs.com/)

## Install required packages

```bash
npm install
```

## Start the web application on your local machine

```bash
npm start
```

You can see it on <http://localhost:3000/>.
